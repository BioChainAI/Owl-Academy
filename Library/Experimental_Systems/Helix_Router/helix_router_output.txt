FINDINGS (measured):
 • Routing Oracle Bias ε = +12.40 > 0, many σ — the geometric oracle decisively
   BEATS random search at equal budget. This is the architecture working as intended:
   the objective is a function of the embedding's features, so geometric proximity is
   a genuinely useful candidate filter — the OPPOSITE of the mod-p case (ε≤0) where it
   could not see the hidden structure.
 • Compressed past-data prior (12× compression of the persistent terrain): NOT a
   significant lift here — it stays within the 2σ band at every observation level. It
   does show a clean monotonic trend (mildly NEGATIVE when all rewards are seen, rising
   toward positive as observation gets sparse), matching the principle that a prior
   helps only when present data is missing — but it never clears significance in this
   regime. Honest read: the geometry is the real win; stored history is marginal here.
 • Helix B keeps every reported route value EXACT — the oracle is never trusted
   blindly. Same safety discipline as the factoring bench; here, a useful purpose.
