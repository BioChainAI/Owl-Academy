SHD-CCP Seed Schema · Crystallization · Torsion Flow
SHD-CCP Seed Schema — Each engram is seeded by a 64-bit integer packed from bit fields: ST/PA/CO/FR/SP/AM. The Mulberry32 PRNG streams these into panmagic 8×8 boolean grids.

XOR Crystallization — Text is tokenized character-by-character; each character grid is XOR'd into an accumulator, building a unique topological fingerprint ("knot") for any sequence.

Chiral Key Format — O1::rootChar::hexhash where hexhash is the 8-byte little-endian pack of the final 64-bit crystallized state.

Torsion Flow J — Computed as (upper_triangle_sum − lower_triangle_sum) / 32 from the active Gamma packet grid. Positive J → exploratory; negative J → grounded; near-zero → balanced.

3-Packet Window — Alpha (oldest), Beta (mid), Gamma (newest, active for resonance). On each utterance: Alpha ← Beta ← Gamma ← new crystallized engram.
