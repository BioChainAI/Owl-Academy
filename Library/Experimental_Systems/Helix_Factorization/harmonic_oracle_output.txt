FINDINGS (measured):
 • Path-length linearizes (R²≈1.000) — so the geometric feature ≈ the index, and
   c only enters as a global constant warp (same for every pair). A global warp
   cannot align the embedding with the mod-p congruence structure of the orbit.
 • Measured Oracle Bias ε is within noise of 0 (no acceleration).
   CRT view of WHY: x_i mod N ↔ (x_i mod p, x_i mod q). A useful collision needs
   x_i ≡ x_j mod p; such pairs differ by a multiple of p and are spread across
   [0,N) — NOT close in any value/harmonic metric. The embedding sees x_i mod N,
   which mixes both CRT parts, so proximity can't isolate the mod-p part.
