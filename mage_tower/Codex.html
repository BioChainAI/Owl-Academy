How to add a minor tome

▶ Video Tome — Go to Prereq Math Hub → Submit Video. Paste any YouTube, Vimeo, or direct URL. It is immediately signed to your Genesis ID and visible to the community.

📄 HTML Lesson — Create an HTML file under Library/Community & Research/Independent_Learning/, then add a single entry to modules.json in that same folder with its id, title, subject, path, and difficulty. On next page load it auto-registers and appears in the hub.

Minor tomes carry no negative consequence — they are either useful or not. Each ★ useful stamp a peer gives earns you +5 XP. Archons and Instructors can further certify exceptional tomes for departmental recognition.
