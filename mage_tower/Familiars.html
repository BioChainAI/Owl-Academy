Sigil Composition Protocol

The user sigil is a deterministic composite SVG generated from three sources: the genesis seed (cosmologicalId), the user's tier, and earned achievements. Every layer is algorithmically derived — no choices are made, only revealed.

Slot System

Slot 0 is your genesis familiar — always bound. Each Echo Familiar purchase reveals the next draw from your seed's PRNG stream. Slot 1 = second draw, slot 2 = third, and so on. Released slots are permanently consumed; their slot number is never reused. Maximum 6 slots (0–5).

Manifold Forge

Each echo slot (slot 1+) may have an independent Hilbert manifold — a nested polygon string-art field rendered behind the familiar sigil. Parameters N (points), K (string multiplier), and T (tiers) control its geometry. Slot 0 always uses a clean background unless explicitly configured.

PRNG Seeding

The xmur3 + sfc32 algorithm seeded with your cosmologicalId produces a deterministic float stream. Slot N burns N draws then takes the Nth+1 as the archetype index. The same ID, same slot, always yields the same familiar.
